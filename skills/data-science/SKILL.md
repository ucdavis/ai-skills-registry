# Data Science

This skill covers data science best practices.

## Language Detection

- `*.py`, `*.ipynb` → read `python/instructions.md`
- R, Julia, or other → apply notebook structure and reproducibility principles from general knowledge

## Universal Principles

- Reproducibility is a first-class concern: pin versions, set seeds, track experiments.
- Validate data before trusting it.
- Keep notebooks for exploration; move reusable logic to Python modules.
- Document assumptions and data lineage.
